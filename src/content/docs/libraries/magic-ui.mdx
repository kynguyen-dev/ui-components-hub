## Overview

**Magic UI** is a collection of animated React components designed for marketing and landing page surfaces. Where shadcn/ui focuses on functional UI primitives, Magic UI focuses on **visual impact** — shimmer effects, particle animations, gradient flows, and text reveals.

## Available Components

| Component | Category | Status | Description |
| :--- | :--- | :--- | :--- |
| **Buttons** | Actions | Stable | Premium animated button patterns including shimmer, rainbow, ripple, shiny, pulsating, and interactive hover styles |
| **Device Mocks** | Showcase | Stable | Browser and mobile device mockups for screenshots, demos, and launch visuals |
| **Backgrounds** | Decorative | New | Animated grid, pattern, ripple, and texture effects for hero sections and ambient page framing |
| **Text Animations** | Typography | Stable | Animated text treatments for hero copy, storytelling sections, stats, and expressive headlines |

### Animated Grid Pattern

Animated square grids that sweep or pulse across the page, useful for launch sections and product hero backdrops.

### Retro Grid

Perspective grid lines with a synthwave-inspired horizon treatment, ideal for bold hero art and demo splash screens.

### Ripple

Expanding concentric wave backgrounds that create motion around a focal callout or centered CTA.

### Dot Pattern

Low-contrast dotted fields that add structure without overpowering cards, docs layouts, or product screenshots.

### Grid Pattern

Clean technical grid lines for editor-inspired interfaces, blueprint motifs, or documentation surfaces.

### Hexagon Pattern New

A honeycomb-style geometric field for more stylized tech branding and dense decorative backgrounds.

### Striped Pattern New

Directional striped layers that create momentum and work well behind badges, stat blocks, or hero copy.

### Interactive Grid Pattern

Pointer-reactive grid backgrounds that add subtle motion as users move through a hero or showcase section.

### Light Rays New

Soft ray-based gradients that simulate volumetric light and work well behind screenshots or centered messaging.

### Noise Texture

Fine-grain texture overlays for breaking up flat color fills and adding a warmer, less synthetic finish.

## Installation

```bash
npx magicui add shimmer-button
```

Or copy the component source directly from the [Shimmer Button docs](/docs/magic-ui-shimmer-button).

## Design Philosophy

- **Motion with purpose** — Animations guide attention to key actions, not distract from them.
- **Performance** — Uses CSS animations and `transform-gpu` for hardware acceleration.
- **Theme-aware** — Components adapt to light/dark mode via CSS custom properties.
- **Composable** — Magic UI components accept `className` and `children` props for full customization.

## When to Use Magic UI

✅ **Use for**: Landing page CTAs, product tours, marketing hero sections, sign-up prompts.

❌ **Avoid for**: Dense data tables, admin dashboards, form-heavy workflows (use shadcn/ui instead).
