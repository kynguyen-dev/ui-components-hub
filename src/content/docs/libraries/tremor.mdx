## Overview

**Tremor** is a React library for building modern dashboards. It provides pre-built components for data visualization, metrics display, and analytics interfaces — all styled with Tailwind CSS.

The hub currently features Tremor's metric card components with plans to add charts, bar lists, and tracker components.

## Available Components

| Component | Category | Status | Description |
| :--- | :--- | :--- | :--- |
| [Metric Card](/docs/tremor-metric-card) | Cards | Stable | KPI card with trend indicators |
| Area Chart | Charts | Planned | Time-series area visualization |
| Bar List | Lists | Planned | Horizontal bar ranking list |
| Tracker | Status | Planned | Status dot timeline tracker |

## Installation

```bash
npm install @tremor/react
```

Or use the hub's standalone metric card component from the [Metric Card docs](/docs/tremor-metric-card).

## Design Principles

- **Data-first** — Components prioritize at-a-glance readability of numbers and trends.
- **Color-coded feedback** — Green for positive, red for negative, neutral for flat — universally understood.
- **Hover interactions** — Subtle hover effects invite exploration without adding visual noise.
- **Dense but clean** — Dashboard components pack information without feeling cluttered.

## When to Use Tremor

✅ **Use for**: Analytics dashboards, admin panels, KPI displays, monitoring interfaces, financial reports.

❌ **Avoid for**: Marketing landing pages (use Magic UI), content-heavy docs (use shadcn/ui), decorative elements.
