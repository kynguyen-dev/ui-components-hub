## Overview

**shadcn/ui** is a collection of beautifully designed, accessible components built on top of Radix UI and styled with Tailwind CSS. Unlike traditional component libraries, shadcn/ui gives you the source code — components are copied into your project, not imported from a package.

## Installation

Add components individually using the CLI:

```bash
npx shadcn@latest add button
npx shadcn@latest add card
npx shadcn@latest add tabs
npx shadcn@latest add command dialog
```

Components are installed to `src/components/ui/` and can be customized freely.

## Design Principles

- **Copy/paste ownership** — You own the code. No external dependency to upgrade.
- **Radix primitives** — Built on Radix UI for accessibility (ARIA, keyboard navigation, focus management).
- **CSS variables** — All colors use CSS custom properties for easy theming.
- **Tailwind-first** — Styled with Tailwind utilities, composable with `cn()`.
