## Overview

This page collects the full **Magic UI text animation** set currently listed in the official components catalog:

- `Text Animate`
- `Typing Animation`
- `Line Shadow Text`
- `Aurora Text`
- `Video Text`
- `Number Ticker`
- `Animated Shiny Text`
- `Animated Gradient Text`
- `Text Reveal`
- `Hyper Text`
- `Word Rotate`
- `Scroll Based Velocity`
- `Sparkles Text`
- `Morphing Text`
- `Spinning Text`
- `Text Highlighter`
- `Text 3D Flip`

All components on this page were added from the official Magic UI registry and rendered locally from `src/components/ui`.
