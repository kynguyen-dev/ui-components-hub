## Overview

Buttons are the most frequently used interactive elements. The hub aggregates button components from **shadcn/ui** and **Magic UI**, giving you options ranging from minimal to high-impact.

[Official Documentation ↗](https://ui.shadcn.com/docs/components/button)

## When to Use Which

| Button | Best For | Library |
| :--- | :--- | :--- |
| Default | Forms, dialogs, standard actions | shadcn/ui |
| Outline | Secondary actions, cancel buttons | shadcn/ui |
| Ghost | Navigation, toolbar items | shadcn/ui |
| Shimmer | Landing page CTAs, sign-up prompts | Magic UI |

## Expanded Comparison

| Button | Best For | Library |
| :--- | :--- | :--- |
| Default | Forms, dialogs, standard actions | shadcn/ui |
| Shimmer | Landing page CTAs, sign-up prompts | Magic UI |
| Borderless/Flat | Content-heavy dashboards | NextUI / Tremor |
| Semantic Sets | Rapid prototyping, distinct alert states | daisyUI |
| Neo-brutalism | Creative portfolios, stylized web apps | cult-ui / kokonut UI |
