## Overview

The **Shimmer Button** is a high-impact call-to-action component from the Magic UI family. It features a rotating conic-gradient shimmer that traces the button border, creating a premium "glow" effect that draws the user's eye without being distracting.

Use this component for:

- Primary CTAs on landing pages
- Sign-up or upgrade prompts
- Any action that needs extra visual weight

## Installation

```bash
npx magicui add shimmer-button
```

## Props

| Prop             | Type     | Default     | Description                              |
| :--------------- | :------- | :---------- | :--------------------------------------- |
| `shimmerColor`   | `string` | `#ffffff`   | Color of the shimmer highlight           |
| `shimmerSize`    | `string` | `0.05em`    | Width of the shimmer beam                |
| `borderRadius`   | `string` | `100px`     | Border radius of the button              |
| `shimmerDuration`| `string` | `3s`        | Duration of one full shimmer rotation    |
| `className`      | `string` | —           | Additional Tailwind classes              |
| `children`       | `node`   | —           | Button content                           |

## Usage Notes

- The shimmer animation uses `conic-gradient` with CSS custom properties for full customizability.
- The button includes a `transform-gpu` utility for hardware-accelerated transforms.
- Works beautifully in both light and dark modes.
