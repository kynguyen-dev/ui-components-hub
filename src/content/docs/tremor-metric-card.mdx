## Overview

The **Metric Card** is a staple of the Tremor design system, purpose-built for dashboards and analytics surfaces. It presents a single KPI with optional trend direction (`up`, `down`, `flat`) and a subtitle for additional context.

Key characteristics:

- **Large metric typography** for at-a-glance readability
- **Color-coded trend badges** (green for up, red for down, neutral for flat)
- **Subtle hover effects** that invite interaction without visual noise

## Props

| Prop        | Type                           | Default  | Description                          |
| :---------- | :----------------------------- | :------- | :----------------------------------- |
| `title`     | `string`                       | —        | Label above the metric               |
| `metric`    | `string`                       | —        | The primary value to display         |
| `delta`     | `string`                       | —        | Change percentage (e.g. `+12.5%`)    |
| `deltaType` | `'up' \| 'down' \| 'flat'`    | `'flat'` | Controls trend icon and badge color  |
| `subtitle`  | `string`                       | —        | Additional context below the metric  |
| `className` | `string`                       | —        | Additional Tailwind classes          |

## Design Considerations

- The card uses `border-border` which automatically adapts to light/dark mode via CSS custom properties.
- Hover effects use `group-hover` for coordinated animation across the card surface.
- The decorative bottom gradient bar appears on hover for a premium interaction cue.
